<details>
<summary><strong>🐍 One-time setup: activate the snake contribution animation</strong></summary>

1. Go to your `PalashHere` repo → **Settings → Actions → General** → set Workflow permissions to **Read and write**
2. Go to **Actions tab → New workflow → Set up a workflow yourself**
3. Name it `snake.yml` and paste this:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - uses: actions/checkout@v3
      - name: Generate snake SVG
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

4. Commit the file, then go to **Actions → Generate Snake Animation → Run workflow**
5. Once it completes, the snake animation above will appear automatically ✅

</details>
